# Codex Docs

Bu papka Aidevix loyihasida Codex'dan professional foydalanish uchun markaziy hujjatlar to'plami.

## Tuzilma

- [Handbook](./handbook.md): asosiy qoida, workflow, token tejamkorlik, verification, va multi-agent mezonlari.
- [Templates](./templates.md): copy-paste uchun tayyor prompt shablonlari.
- [Daily Prompts](./daily-prompts.md): kunlik ishlatish uchun eng ko'p kerak bo'ladigan 10 prompt.

## Tez Boshlash

1. Avval [Handbook](./handbook.md) ni o'qing.
2. Ish turiga qarab [Templates](./templates.md) dan prompt oling.
3. Tezkor kundalik ish uchun [Daily Prompts](./daily-prompts.md) dan foydalaning.

## Eng Qisqa Formula

Har promptda quyidagi 4 qism bo'lsin:

```text
Muammo:
[aniq bug]

Scope:
[aniq fayllar]

Kutilgan natija:
[nima ishlashi kerak]

Cheklov:
Minimal patch qil.
Keraksiz refactor qilma.
```

## Eng Foydali Qo'shimcha Satrlar
